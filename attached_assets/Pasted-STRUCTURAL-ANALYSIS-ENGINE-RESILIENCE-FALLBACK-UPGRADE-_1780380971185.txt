STRUCTURAL ANALYSIS ENGINE — RESILIENCE & FALLBACK UPGRADE

OBJECTIVE

Prevent the Entry Checker from becoming mostly unusable when OHLCV candle data is unavailable.

Create graceful degradation instead of complete metric failure.

========================================================
OHLCV FALLBACK HIERARCHY

Current Issue:

The engine relies heavily on GeckoTerminal 15m candles.

If GeckoTerminal OHLCV fails:

- Acceptance Score becomes unavailable
- Demand Return becomes unavailable
- Bounce Velocity becomes unavailable
- Shelf Analysis becomes unavailable
- Confidence Tier becomes severely degraded

Required Change:

Add OHLCV fallback hierarchy.

Attempt candle retrieval in this order:

1. GeckoTerminal OHLCV

2. DEX Screener OHLCV
   (if available)


Only mark candle-dependent metrics unavailable if ALL candle sources fail.

Display:

OHLCV Source:

GeckoTerminal
DEX Screener

========================================================
TRANSACTION PRESSURE SCORE

Purpose:

Estimate buyer dominance without candles.

Inputs:

Buy Count

Sell Count

Buy USD Volume

Sell USD Volume

Primary Formula:

BuyDominance =
BuyVolume /
(BuyVolume + SellVolume)

Score:

≥70% = 100

60–70% = 80

50–60% = 60

40–50% = 40

<40% = 20

Classifications:

Strong Buyer Pressure

Moderate Buyer Pressure

Neutral Flow

Weak Buyer Pressure

Seller Dominated

Display:

Transaction Pressure Score
